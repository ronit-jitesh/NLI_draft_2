## Notes for Teammates

- All CSV files in `results/` are the raw prediction outputs ‚Äî every number in the report can be verified by loading these files and running `sklearn.metrics.accuracy_score` / `classification_report`.
- The `data/` CSVs are the test sets. Seed 42 is used everywhere.
- `PROMPTS.md` contains the exact text of all 5 prompt strategies used in `03_gpt4o_prompting.py` and `04_other_llms.py`.
- If re-running from scratch, notebooks 03‚Äì05f require API keys in `.env`. Estimated cost: ~$8‚Äì10 total.
